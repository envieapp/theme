## Installation

Install @envie/theme with npm

```bash
npm install @envie/theme \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```

Install @envie/theme with yarn

```bash
yarn add @envie/theme \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```
